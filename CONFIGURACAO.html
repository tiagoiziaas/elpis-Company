🚀 Configuração do Banco Elpis

⚠️ Importante: Siga estes passos na ordem para configurar corretamente o banco de dados.

📋 Passo 1: Obter Chaves do Supabase

1. Acesse: https://supabase.com/dashboard/project/gjurkteiuwbdswumuloh/settings/api

2. Copie a Publishable key

3. Copie a service_role key

📋 Passo 2: Obter Senha do Banco

1. Acesse: https://supabase.com/dashboard/project/gjurkteiuwbdswumuloh/settings/database

2. Em "Connection string", a senha está entre colchetes: postgres:[SENHA]@db...

📋 Passo 3: Atualizar .env

Edite o arquivo .env com as chaves obtidas:

NEXT_PUBLIC_SUPABASE_URL=https://gjurkteiuwbdswumuloh.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=(cole sua chave anon aqui)
SUPABASE_SERVICE_ROLE_KEY=(cole sua chave service_role aqui)
DATABASE_URL=postgresql://postgres:(sua senha)@db.gjurkteiuwbdswumuloh.supabase.co:5432/postgres
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=Elpis2026SecretKeyForNextAuthSuperSecure123

📋 Passo 4: Criar Tabelas

1. Acesse: SQL Editor do Supabase

2. Abra o arquivo create-tables.sql e copie TODO o conteúdo

3. Cole no SQL Editor

4. Clique em "Run"

✅ Se aparecer "Success", as tabelas foram criadas!

📋 Passo 5: Popular com Dados

No terminal, execute:

cd C:\Users\Folhatech\Desktop\Koliel\elpis
node seed-db.js
✅ Se aparecer "🎉 Seed completed successfully!", os dados foram inseridos!

📋 Passo 6: Testar

1. Acesse: http://localhost:3000

2. Clique em "Entrar"

3. Use: ana@elpis.com / 123456
